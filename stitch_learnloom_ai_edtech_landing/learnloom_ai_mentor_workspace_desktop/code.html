Can you show me a simple example of how to use useEffect to fetch data when a component mounts?

Certainly! The useEffect hook is perfect for side effects like data fetching. To run it only once when the component mounts, you pass an empty dependency array [] as the second argument.

Here is a concise example:

jsx 
import React, { useState, useEffect } from 'react';

function DataFetcher() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // This runs once when component mounts
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(result => {
        setData(result);
        setLoading(false);
      });
  }, []); // <-- Empty array means "mount only"

  if (loading) return <div>Loading...</div>;
  return <div>{JSON.stringify(data)}</div>;
}
