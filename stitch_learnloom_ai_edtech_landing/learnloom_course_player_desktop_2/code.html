Self-Attention Mechanism

In this lesson, we break down the core innovation of the Transformer architecture: the self-attention mechanism. Unlike RNNs that process sequences step-by-step, self-attention allows the model to look at the entire sequence simultaneously to determine which parts are most relevant to each other.

We will cover the mathematical intuition behind Queries, Keys, and Values, and how the scaled dot-product attention formula computes the attention weights.

Key Takeaways

  • Self-attention solves the vanishing gradient problem in long sequences.
  • Q, K, V matrices are derived from the same input embedding.
  • Scaling by the square root of dimension size prevents softmax saturation.

Here is a basic implementation of scaled dot-product attention:

import torch
import torch.nn.functional as F

def attention(q, k, v, d_k):
    scores = torch.matmul(q, k.transpose(-2, -1)) 
    scores = scores / math.sqrt(d_k)
    weights = F.softmax(scores, dim=-1)
    return torch.matmul(weights, v)
