Absolutely! Think of a neural network like a series of interconnected filters. Here's a very simple conceptual example in Python using a basic activation function.

python 
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# Simple input and weights
inputs = np.array([0.5, 0.8, -0.2])
weights = np.array([0.4, 0.6, 0.1])

# Calculate output
output = sigmoid(np.dot(inputs, weights))
print(f"Output: {output}")
