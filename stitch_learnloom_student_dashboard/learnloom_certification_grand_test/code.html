Implement Advanced Load Balancer

Design and implement a thread-safe load balancer that distributes incoming requests across a set of healthy servers using a weighted round-robin strategy.

Requirements

  • Must implement the ILoadBalancer interface provided.
  • Support dynamic addition and removal of servers with their respective weights.
  • Handle concurrent requests safely without deadlocks.
  • Ensure servers marked as 'unhealthy' are temporarily removed from the pool until a successful health check.

Input Constraints

1 <= Number of Servers <= 1000

1 <= Server Weight <= 100

Maximum concurrent requests: 10,000
